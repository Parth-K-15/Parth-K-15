## Featured Projects

### 🎓 Education Consultancy Platform — [FreeCollegePredictor](https://freecollegepredictor.com) · [ToppersClubs](https://toppersclubs.com) · [MBBSRoutes](https://mbbsroutes.com)
Production MERN platform serving **32,000+ users**, with a CatBoost-based JEE/NEET college prediction engine, 4-tier RBAC (SuperAdmin/Admin/Influencer/Calling Team), and OTP authentication via AWS SNS. Deployed on AWS EC2, S3, and CloudFront with PayU payment tracking and a referral attribution system.
`MERN` `AWS` `CatBoost` `REST APIs`

### 🤖 GenAI-Powered HR Analytics & Workforce Intelligence Platform
Data pipelines processing 4,001 employee records across 25 ML modules and 7 workforce domains — classification, clustering, regression, and statistical testing with scikit-learn. Integrated the Gemini API for automated executive insight generation with retry/backoff and strict JSON schema validation.
`Python` `Scikit-learn` `Gemini API` `Pandas`

### 📝 AI-Based Smart Form Filler
Fine-tuned `bert-base-uncased` on a 27-class form-field classification task, reaching **93.9% accuracy** (vs. 91.97% for a BiLSTM baseline). Combined BERT predictions with rule-based fallbacks and Selenium automation to auto-fill Google Forms, with fuzzy matching via RapidFuzz and a Streamlit control dashboard.
`Python` `BERT` `BiLSTM` `Selenium` `Streamlit`

### 💬 NLP Product Review Analysis — Quantiphi Winner
End-to-end NLP pipeline on 801 Flipkart reviews: scraping, deduplication, translation, and preprocessing. Bi-LSTM sentiment classifier reached **85.19% accuracy**, alongside topic modeling, semantic similarity, and a retrieval-based QA dashboard. Recognized by Quantiphi's SMEs.
`Python` `NLTK` `SpaCy` `TensorFlow` `Bi-LSTM`
